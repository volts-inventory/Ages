## Q21 — Cohort-level baseline observations
**Status:** Decided
**Context:** Initially only named figures generated observations, which
implied named figures start from a blank slate when promoted —
implausible.
**Decision:** Cohort tracking accumulates **summary statistics** per
region per recognized phenomenon (count, running mean, running
variance), *not* full data points. Named figures inherit summary stats
at promotion — they get the aggregate state of their culture, not
synthesised individual data points. Pattern detection and hypothesis
fitting (Q32) still require individual data points sampled from
physics state, which only named figures generate (phase B), so
discoveries still come from named figures.
**Lives in:** `docs/civ.md` Discovery → A.
