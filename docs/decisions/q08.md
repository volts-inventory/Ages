## Q8 — Phenomenon catalog curation
**Status:** Superseded by Q40
**Context:** Where does the master archetype list live and how is it
authored?
**Decision:** Originally: YAML in `data/phenomena/`, schema-validated,
with archetype entries carrying gates, parameter formulas, depth tiers,
relations, true_formula, valid_hypothesis_forms. Superseded by Q40 —
the equivalent in the physics-engine model is a *recognition-template
library* in `data/recognition/`, which describes how to detect emergent
phenomena from physics state (signature + sensorium gates + depth tiers
+ valid_hypothesis_forms). No `true_formula` field: the truth is what
the physics produces.
**Lives in:** `docs/world.md` Recognized phenomena section,
`AGENTS.md` repo layout.
