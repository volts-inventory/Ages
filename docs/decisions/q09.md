## Q9 — Cultural taboo strength
**Status:** Decided (M4: dogmatism-scaled allow_observation hook;
behavioural consumer waits on Q57 settlement-pop for cell-level
suppression)
**Context:** How aggressively does cultural taboo suppress
observation and transmission? M3 ships
`culture_hooks::allow_observation` as a no-op; M4 wires it.
**Decision:** Hook signature stays `allow_observation(template_id,
cell, dogmatism) -> Real` returning a suppression probability
in `[0, 1]` rather than a hard bool — lets callers roll
deterministically against per-cell RNG once Q57 ships per-cell
state. M4-min returns `0.0` (never suppresses); the behavioural
consumer lands when Q57 settlement-pop adds per-cell taboo
state. Hook now reads `dogmatism` from Q24 cosmology so the
formula is in place even without an active consumer.

**Future formula (when Q57 wires):**
```
suppression = clamp(dogmatism × cosmology_aligned_against(template), 0, 0.9)
```
Per-template alignment-against tags ship with Q57 v2.

**Lives in:** `sim/civ/culture_hooks.rs`.
