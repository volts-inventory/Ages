## Q103 — Vertical magnetic-field component (true 3D Lorentz)
**Status:** Decided
**Context:** Q94 promoted the planetary magnetic field from a
scalar stub into a vector `(B_q, B_r)` — but only in the surface
plane. Real planetary dipoles have a *vertical* component too,
which is largest at the magnetic poles (where the horizontal
component vanishes). The Q98 Lorentz law's `q · v × B` cross
product in our 2D model needed *vertical* B for the rotational
deflection of horizontal v; pre-Q103 we used `|B|` (the horizontal
magnitude) as a proxy, which was correct at mid-latitudes but
wrong at the poles where horizontal magnitude → 0 yet B_z is
largest.

The Q98 / Q94 docs both flagged this as a follow-up.

**Decision:**

1. **Add `magnetic_b_z: Vec<Real>`** to `PhysicsState` alongside
   `magnetic_b_q` and `magnetic_b_r`. New accessors
   `magnetic_field_z()` and `magnetic_field_z_mut()` (in
   addition to the existing 2-tuple `magnetic_field()` / `_mut`).
2. **`Magnetism::init_field` and `::integrate`** now write all
   three components. The vertical component follows real-dipole
   physics:
   ```
   |B_z|  = 2 · dipole_strength · sin(latitude)
   sign(B_z) = +1 in northern hemisphere, -1 in southern, 0 at equator
   ```
   The factor of 2 reflects the real-dipole pole-to-equator
   ratio: at the poles `|B| = 2 · dipole`, at the equator
   `|B| = dipole`, total `|B| = dipole · sqrt(1 + 3 sin²(lat))`.
3. **`magnetic_field_magnitude`** cache now includes `B_z`:
   `sqrt(B_q² + B_r² + B_z²)`. Recognition templates that key on
   `Field::MagneticMagnitude` (e.g. `magnetic_field_strong`)
   now correctly fire stronger at the poles.
4. **Q98 `Lorentz::integrate`** reads `state.magnetic_field_z()`
   directly instead of the magnitude. The 2D cross product
   `(v_q, v_r, 0) × (0, 0, B_z)` gives:
   ```
   Δv_q = +coeff · q · v_r · B_z
   Δv_r = -coeff · q · v_q · B_z
   ```
   Sign of `B_z` flips between hemispheres, so the deflection
   rotates the wind in opposite senses across the equator —
   exactly the pattern the upcoming Q106 Coriolis force will
   reinforce.

**Why this respects the project goals.**
- *Physics as close to real as possible:* the Lorentz force
  on horizontal velocity now uses the *correct* vertical
  field component, not a magnitude proxy. Polar deflection is
  strongest (where real planets see auroral currents); equatorial
  deflection is zero (where real planets see no Lorentz
  contribution to horizontal flow).
- *Total |B| pattern matches real planets:* poles have the
  strongest field (factor 2× equator), equator the weakest.
  Recognition templates that fire on field-magnitude thresholds
  now have the correct geographic distribution.
- *Sign-flip across hemispheres:* the southern-hemisphere
  Lorentz deflection is the mirror image of the northern, so
  charged-air drift in the two hemispheres is mirror-symmetric.
  Q106 Coriolis will produce the same hemisphere-mirroring for
  uncharged air, completing the geostrophic story.
- *Determinism:* same write pattern as Q94; new `B_z` field is
  deterministic from latitude only; no per-tick allocation
  beyond a single per-cell `mag_after` buffer.

**Numerical care.**
- `B_z` peaks at `2 · dipole_strength = 100` for `Strong`
  magnetosphere — well inside Q32.32.
- `sqrt(B_q² + B_r² + B_z²)` peaks at `sqrt(0² + 0² + 100²) =
  100` at the poles — the recognition `magnetic_field_strong`
  threshold of 20 still fires across most of the planet.
- The `match signed_offset.cmp(&0)` for the sign branch
  avoids clippy's `comparison_chain` warning.

**Lives in:** `sim/physics/src/state.rs` (`magnetic_b_z` field
+ accessors + cache update),
`sim/physics/src/magnetism.rs` (real-dipole `B_z` formula in
both `init_field` and `integrate`; updated tests
`q103_horizontal_strongest_at_equator` /
`q103_total_magnitude_strongest_at_poles`),
`sim/physics/src/lorentz.rs` (reads `B_z` directly; updated
test moves charge to a non-equatorial cell).

**Follow-ups (not blocking).**
- *Off-axis dipole.* Real magnetic axes tilt ~10° from the
  rotational axis. Lands when `Planet` carries a magnetic-axis
  tilt field separate from the rotational one.
- *Multipole structure.* Beyond the dipole, real fields have
  quadrupole and higher components (Earth's South Atlantic
  Anomaly). Far-future refinement.
- *Aurora template refinement.* The existing
  `aurora_polar` (id 35) and `magnetic_field_strong` (id 6)
  now fire correctly at high-`|B|` polar cells. A dedicated
  "polar aurora" template combining `B_z > T1` with `Charge
  > T2` would catch the real aurora signature precisely.
