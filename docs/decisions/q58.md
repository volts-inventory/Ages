## Q58 — Literacy mechanic
**Status:** Decided (M4-minimal: discovery-rate proxy with
persistence-tier modifier; bounded-language scoring deferred)
**Context:** PLANNING M3 + Q3 + Q56 reference "literacy" as a
civ-level scalar but never define it. Q3 figure-cap formula
mentions `population × literacy × institution_count`; Q56's
unlock trigger asks for `literacy_floor` per tool. Without a
literacy definition, those formulae can't fire.

True literacy in the project's vision (M4 bounded-language) is
the fraction of the population that can encode and decode the
civ's own symbol system. M4 doesn't yet have an authored
symbol-system mechanic, so v1 ships a tractable proxy: the civ
becomes more "literate" as it accumulates discoveries and as
its persistence tier matures, capturing the empirical fact that
real-world literacy rises with documented knowledge × durable
storage.

**Decision:** `civ.literacy: Real` in `[0, 1]`, computed each
tick from current civ state:

```
literacy = sigmoid_lite(
    DISCOVERY_WEIGHT × n_unique_confirmed_relations
  + TIER_WEIGHT × persistence_tier_index
  + LIFESPAN_WEIGHT × ticks_since_founded / LIFESPAN_DENOM
)
```

`sigmoid_lite(x) = x / (1 + |x|)` is a cheap bounded-output
saturating map (asymptotic to 1, well-defined under Q32.32 — no
exp needed for this bound). Per Q28's umbrella the placeholder
weights are:

- `DISCOVERY_WEIGHT = 0.04` (25 unique discoveries → ≈0.5
  literacy contribution from this term alone)
- `TIER_WEIGHT = 0.20` (each Q28/Q30 persistence tier adds
  meaningfully; T2 written ≈ +0.4)
- `LIFESPAN_WEIGHT = 0.10` (long-running civs accrete literacy
  even without breakthroughs — sustained tradition matters)
- `LIFESPAN_DENOM = 500` (ticks)

Persistence-tier index is the civ's current Q28/Q30 tier
(currently a derived from tech_multiplier proxy until M4 v2
authored persistence tiers land — for v1 use `tier_index = 0`
since M3 ships with no persistence-tech wiring).

**Why this is the right minimal:**
- Single Real on Civ; cheap to compute per tick.
- Monotonic: confirmed relations only adds; tier only goes up;
  age only grows. Literacy never decreases (collapse zeros it
  via successor civ being fresh).
- Bounded `[0, 1]`; no overflow risk.
- Trivially deterministic — pure function of integer counters.

**Wired here:**
- `Civ::literacy_score()` returns the value.
- Q56 reads it as the unlock-floor gate.
- Q3 figure-cap formula consumes it once roster-growth lands
  (currently the founding band is the only roster).

**Deferred to v2:**
- Real bounded-language scoring (M4 phoneme + grammar mature
  enough to track per-individual literacy fraction).
- Authored Q28/Q30 persistence-tier transitions.
- Literacy decay during cultural-lock (Q9).

**Lives in:** `sim/civ/lib.rs` `Civ::literacy_score()`,
`sim/civ/tech.rs` Q56 trigger.
