## Q25 — Focus priority dynamics
**Status:** Decided (M4: cosmology-weighted focus_weight hook;
fit-cadence consumer wires in M4 v2)
**Context:** A civ-level cultural priority vector biases named-
figure focus; M3 ships `culture_hooks::focus_weight` as a
constant 1.0.
**Decision:** Hook reads the civ's cosmology vector (Q24) and
returns a per-relation weight. M4-min formula:

```
focus_weight(relation_id, civ) = 1.0 + EMPIRICAL_BONUS × cosmology.empirical
                                     + REFORMIST_BONUS × cosmology.reformist
```

with `EMPIRICAL_BONUS = REFORMIST_BONUS = 0.25`. Empirical
civs weight all relations more (more attention to the world);
reformist civs likewise. The relation_id parameter is unused
in v1 (no per-relation taxonomy yet); v2 adds per-relation
"novelty class" so reformist civs specifically prioritise
novel forms.

**Behavioural consumer:** the hypothesizer multiplies per-
relation `attempt_period` by `2.0 - focus_weight` (capped at
`[1, 100]`) so a high-focus civ retries fits more often. Wires
when M4 v2 lands the per-relation cadence variation.

**Lives in:** `sim/civ/culture_hooks.rs`.
