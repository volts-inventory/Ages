**Status:** Superseded by Q40

`data/phenomena/`; M1 fills with ~30 archetypes. Superseded by Q40 —
there is no `data/phenomena/`. Equivalent under Path D is
`data/recognition/` containing recognition-template YAML files. M0

with the first ~25 templates as the recognition layer lands.
