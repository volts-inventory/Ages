## Q5 — Phenomenon catalog size
**Status:** Superseded by Q40
**Context:** Originally proposed ~40 archetypes for M1, growing to ~80
by M2.
**Decision:** Replaced first by the master-catalog-with-filtering model
(Q17), then by Q40's pivot to a physics engine. There is no authored
phenomenon catalog any more; phenomena emerge from physics, identified
by a small library of recognition templates.
**Lives in:** `docs/world.md` Recognized phenomena section.
