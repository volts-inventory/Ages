# Q159 — Religion / customs as the fast cultural-divergence layer

**Status:** Decided.

## What

Add a new three-axis `Religion` vector to every civ, sitting on top
of the existing five-axis `Cosmology`. The Q158 belligerence kinship
calc now reads both layers, weighted toward religion as the dominant
signal so intra-species civs can actually reach low kinship and fire
wars.

```
Religion {
    theology    ∈ [-1, 1]   monist ↔ pluralist
    ritual      ∈ [-1, 1]   pragmatic ↔ liturgical
    sacred_time ∈ [-1, 1]   cyclical ↔ eschatological
}
```

Founding-time religion is **not species-anchored** (unlike cosmology
which inherits Q172's species bias). It's set at civ founding from:

```
theology    = clamp(-0.5·doubt    + jitter(civ_id, 1), [-1, 1])
ritual      = clamp( 0.5·charisma + jitter(civ_id, 2), [-1, 1])
sacred_time = clamp(-0.3·curiosity + jitter(civ_id, 3), [-1, 1])
```

`jitter` is a deterministic civ_id-keyed splitmix-style hash mapping
to ±0.20 per axis. So even two civs of the same species with
identical-seeming founders land on different religion vectors — the
designed source of intra-species cultural divergence.

## Why

Q158 introduced belligerence-driven war, but the kinship dampener
read mostly cosmology. Single-species runs have every civ inherit
`species.initial_cosmology` (Q172), so the cosmology gap stays near
zero across every pair through the run, kinship sits at ~1.0, and
the dampener crushes belligerence below the war threshold. We had
to crank `KINSHIP_DAMPENER` down to 0.20 just to get any wars.

The deeper bug: what the sim was calling "cosmology" — empirical /
communitarian / reformist / mystical / hierarchical — isn't really
cosmology in the strict sense (no time-orientation, no reality-
ontology). It's closer to **cultural-religious dispositions**
(Hofstede-style values), which is exactly the layer that diverges
fast in real history within a generation via founding figures,
catastrophe responses, and geography — and is the proximate cause
of war (Reformation, sectarian conflict, "clash of civilizations"
patterns).

Q159 splits this into two layers explicitly:

- **Cosmology** stays as-is in shape (5 axes, Q172 species bias)
  but is now the *slow-drift deep worldview* layer. Its `push_for_*`
  magnitudes were halved and its emit threshold raised from 0.20
  to 0.50, so it moves on near-millennium timescales.
- **Religion** is the new fast layer. Same `push_for_*` table
  shape with **3× the magnitude** of cosmology's reduced values
  (so net religion drift is ~6× cosmology drift per event). Founds
  at civ-genesis time with figure-trait-driven divergence + jitter,
  drifts on the same hypothesis + lifecycle events, emits
  `ReligionShifted` at a 0.20 L2 threshold.

The Q158 kinship calc weights now reflect this:

```
kinship = 0.10·(1 − hier_gap)
        + 0.15·(1 − cosmo_gap)
        + 0.15·(1 − tech_gap)
        + 0.60·(1 − religion_gap)         ← new dominant term
```

Religion at weight 0.60 means even saturated single-species civs
can have kinship dispersed across [0.4, 0.95] by tick 5000+ (the
religion vector founds with jitter then drifts on hypothesis events).

## Drift-hook split

Every existing cosmology drift call site now applies *both* a
cosmology push and a religion push in the same conceptual direction:

| Event                       | Cosmology push (Q24, halved)   | Religion push (Q159)            |
|-----------------------------|--------------------------------|---------------------------------|
| `RelationConfirmed`          | empirical+, mystical−, reformist+ | theology−, sacred_time+        |
| `RefinementProposed`         | empirical+, reformist+         | ritual−, sacred_time+           |
| `RefinementConfirmed`        | empirical+, reformist+, mystical− | theology−, ritual−, sacred_time+ |
| `RefinementRejected`         | empirical−, reformist−, mystical+, hierarchical+ | theology+, ritual+, sacred_time− |
| `MeasurementConfirmed`       | (same as RelationConfirmed)    | (same as RelationConfirmed)     |
| `Falsified`                  | (same as RefinementProposed)   | (same as RefinementProposed)    |
| `Revalidated`                | (same as RelationConfirmed)    | (same as RelationConfirmed)     |
| `Lapsed`                     | (same as RefinementRejected)   | (same as RefinementRejected)    |
| `CivCollapsed`               | communitarian+, reformist−, mystical+, hierarchical+ | theology+, ritual+, sacred_time+ |

Both pushes scale by founding-figure charisma (Q59), same as
before — so charismatic figures still dominate cultural impulses.

## Wire events

New variant on `protocol::Event`:

```rust
ReligionShifted {
    tick: u64,
    civ_id: u32,
    axes_q32: Vec<i64>,          // [theology, ritual, sacred_time]
    dogmatism_q32: i64,          // L2-norm / sqrt(3), clamped
}
```

`CosmologyShifted` payload shape unchanged; only its emission gate
threshold rose (0.20 → 0.50).

## What didn't change

- `Cosmology` struct shape + its 5 axes
- `Cosmology` field on `Civ` + `apply_cosmology_push`
- Q24 hooks fire from the same call sites
- Q26 `form_distance` (still reads cosmology only — affects which
  hypothesis forms a civ favours, separate from inter-civ war)
- `is_peaceful_pair` for diffusion gating (still cosmology-based)
- Q172 species cosmology bias — religion has *no* species bias by
  design

## Calibration

Initial smoke-tune target: 5–15 wars per 8000-tick / seed-100 run
(up from Q158's 2 wars). Achieved by:

- `KINSHIP_W_RELIGION = 0.60` — religion as dominant kinship term
- Religion `push_for_*` magnitudes 3× reduced cosmology magnitudes
  so per-event religion impulse ≈ 6× pre-Q159 cosmology impulse
- Founding `jitter` range ±0.20 per axis — small enough to keep
  nascent civs near-kin if their figures are similar, large enough
  to give civ_id variation real bite

Q158's `KINSHIP_DAMPENER` stays at 0.20 — religion supplies enough
inter-civ dispersion that the dampener doesn't need to be cranked
back up.

## Open follow-ups

- Terrain-driven religion: coastal civs lean reformist+pragmatic,
  mountain civs lean liturgical+monist, plains lean communitarian.
  Currently the only spatial signal is the founding figure's
  `cell_assignment`; threading terrain habitability into the
  founding offset would tie religion to geography (river-flood
  cyclical-time, mountain hierarchy etc.).
- Religion-targeted catastrophe response: a volcanic eruption near
  a civ's apparatus cell could push religion harder than a generic
  collapse — currently every CivCollapsed gives the same impulse.
- Religion-aware diffusion suppression: pair-level religion gap
  might gate `KnowledgeDiffused` similarly to the existing
  cosmology-hierarchy floor in `is_peaceful_pair`.
