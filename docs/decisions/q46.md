## Q46 — Post-run report contents
**Status:** Open (M6)
**Context:** With UI and LLM narration both dropped (Q39, Q51), the
post-run report is the only after-the-run output. Live CLI streaming
covers the during-the-run experience. The report's structure
determines what a reader takes away from a finished run.
**Decision:** Pending. Initial draft of report contents: civ history
timeline, discoveries with fitted equations and confirmation
residuals, named-figure cards, communication and persistence ladder
progression with derived forms, population trajectory, "interesting
moments" highlight reel (Q13), run termination cause (per Q12
taxonomy).
**Recommended starting position:** Sections — planet card, species
card, run-summary stats (recognized-phenomena coverage, civ count,
total figures, run-end cause), per-civ chapters (founding event, key
discoveries with fitted equations, named-figure cards, persistence /
reach ladder progression with derived forms, collapse cause),
inter-civ knowledge-transmission events, species population timeline,
"interesting moments" highlight reel (Q13). Markdown default;
templated prose where prose helps readability — fully deterministic,
no LLM. HTML render via template if desired.
**Lives in:** `docs/architecture.md` Post-run report section.
