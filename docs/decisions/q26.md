## Q26 — Hypothesis suppression strength
**Status:** Decided (M4: confidence-multiplier hook wired into
fit pipeline; cosmology-distance to form uses Q24 axes +
form-tag alignment table)
**Context:** A hypothesis contradicting cosmology accumulates
confidence more slowly; M3 ships
`culture_hooks::suppress_confidence` as a pass-through.
**Decision:** Hook signature becomes
`suppress_confidence(form, raw_confidence, civ) -> Real`
returning the multiplier `[0, 1]`. Wired into the fit pipeline:
`Hypothesizer::best_confirmable_fit` multiplies the raw fit
confidence by this factor before checking
`is_confirmed (>= exp(-1))`. Heretical forms in dogmatic civs
fail to confirm.

**Formula:**

```
suppress_factor = 1 - dogmatism × form_distance(form, civ.cosmology)
```

`form_distance` is a per-form-class distance to the civ's
preferred form-class derived from cosmology axes:

| Form class | Preferred when ... | Distance from this preference |
|---|---|---|
| `Constant` | high `Tradition` (low `Reformist`) | `(1 + reformist) / 2` |
| `Linear` | neutral | `0` (always 0) |
| `ThresholdStep` | high `Empirical` | `(1 - empirical) / 2` |
| `Polynomial2/3` | high `Reformist` AND high `Empirical` | `(1 - (reformist + empirical)/2) / 2` |
| `PeriodicSine` | high `Mystical` | `(1 - mystical) / 2` |
| `ExpDecay/Growth` | high `Empirical` AND low `Tradition` | `((1 - empirical) + reformist) / 4` |
| `Logistic` | high `Reformist` | `(1 - reformist) / 2` |
| `PowerLaw` | high `Empirical` AND high `Reformist` | `(1 - empirical + 1 - reformist) / 4` |
| `Logarithmic` | high `Empirical` | `(1 - empirical) / 2` |
| `InverseSquare` | high `Empirical` | `(1 - empirical) / 2` |

All distances clamp to `[0, 1]`. A civ at neutral cosmology
gets distance 0 for every form (no suppression). A
high-Mystical civ finds Linear / ThresholdStep / Polynomial /
ExpDecay / PowerLaw / Logarithmic / InverseSquare all
"distant" and suppresses them — only PeriodicSine + Constant
escape full suppression. That's the right narrative shape:
mystical civs prefer cyclic / static cosmologies over
mechanistic ones.

**Lives in:** `sim/civ/culture_hooks.rs` (formula),
`sim/civ/discovery.rs` (consumer in `best_confirmable_fit`),
`sim/civ/cosmology.rs` (form-distance helper).
