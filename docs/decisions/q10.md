## Q10 — Conflict resolution math
**Status:** Decided (M5: per-pair periodic check on overlapping
claims; cosmology-derived conflict-norm; population × literacy
strength model)
**Context:** When civs claim overlapping cells (M5 concurrent
multi-civ via Q48-v3 breakaway), how is the contest decided?
The current implementation lets two civs co-exist on the same
cells indefinitely, which is unrealistic. Q10 makes the
relationship dynamic.

**Decision:** Two-phase model — periodic conflict + peaceful
diffusion when no conflict.

**Conflict resolution.** Every `CONFLICT_CHECK_TICKS = 75` ticks
(Q67: ticks are now months, so this fires roughly every 6 years —
intentionally unscaled so border conflicts happen at a realistic
cadence rather than once per 75 years), for each pair of active
civs whose `claimed_cells` overlap by at least one cell, run a
contest:

```
strength(civ) = aggregate_population × (1 + literacy_score)
              × (1 + Hierarchical / 2)        // war-cosmology bonus
contest_winner = whichever strength is higher (deterministic tie-break by id)
loser_pop_loss_fraction = clamp(0.10 + Hierarchical_winner × 0.30, 0, 0.50)
```

The Hierarchical axis from Q24 cosmology drives the conflict
intensity: high-Hierarchical civs (warlike) inflict more
damage; low-Hierarchical (egalitarian) civs only peck at each
other.

After resolution: loser's region cohorts in the disputed cells
shrink by `loser_pop_loss_fraction`; if loser's aggregate falls
below `CONFLICT_DEFEAT_FLOOR = 50`, loser surrenders the
disputed cells (drops them from `claimed_cells`); the winner
takes over those cells (adds to `claimed_cells`). Otherwise
the contest continues next cycle. `ConflictResolved` event
fires per resolution.

**Knowledge diffusion** (peaceful contact, no conflict). Every
`DIFFUSION_CHECK_TICKS = 100` ticks (Q67: ~8 years under tick =
month, also intentionally unscaled), for each pair of active
co-existing civs whose conflict-norm is *peaceful* (both
civs' `Hierarchical < 0.4`), comprehend a fraction of the
other civ's confirmed relations into the receiver's first
figure. Comprehension uses Q50's formula but without the age-
decay (artifacts aren't in transit; this is direct contact).
`KnowledgeDiffused` event fires per transmitted relation.

**Why these specific thresholds:**
- 75 ticks for conflict gives time for cosmology + population
  shifts to register; 100 for diffusion is slower so peaceful
  contact yields meaningful but not instant convergence.
- 0.4 Hierarchical floor for peaceful contact: the project goal
  "different worlds, different sciences" needs civs to *not*
  homogenize too fast; the gate keeps mid-Hierarchical pairs
  in conflict mode.
- 0.50 max loss: civs don't get one-shot annihilated; multiple
  contest cycles produce the realistic-feeling slow decline.

**Lives in:** `sim/civ/conflict.rs` (strength + outcome math),
`sim/civ/transmission.rs` (peaceful diffusion alongside Q50
transmission), `sim/core::CivLifecycle` (conflict trigger),
`sim/core::CulturalDrift` (diffusion trigger),
`protocol::ConflictResolved` + `protocol::KnowledgeDiffused`.
