## Q17 — Master catalog with per-planet filtering
**Status:** Superseded by Q40
**Context:** Should each milestone have its own catalog, or one growing
catalog filtered per run?
**Decision:** Originally: one master catalog filtered twice (physical-
presence, then sensorium). Superseded by Q40 — there is no archetype
catalog. The two-pass filtering pattern survives, applied to the
recognition-template library: physical-presence pass = which templates'
signatures fire in this run's physics; sensorium pass = which of those
the species can detect, with latent vs perceivable-now split.
**Lives in:** `docs/world.md` Recognized phenomena section + Run start
order.
