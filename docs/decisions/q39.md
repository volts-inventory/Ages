## Q39 — Drop UI
**Status:** Decided
**Context:** Original plan included M5–M8 building a Godot 4 viewer
(text viewer → tilemap → scrubber → knowledge-graph). UI work scoped
~4–8 weeks of focused dev time. The simulation has value standalone;
the UI was an accessory.
**Decision:** Drop the live UI entirely. Output media for human
consumption become: NDJSON event log (machine-readable, M1+), live
CLI event stream during the run (human-readable, M1+, with verbosity
levels including `--cli=highlights` once Q13 lands), post-run report
(markdown, M6). LLM narration also dropped (see Q51). A web viewer
can return as a small project later if the sim warrants one. Freed
scope flows into Q40's physics engine and deeper culture / multi-civ
work.
**Lives in:** `PLANNING.md` Milestones, `docs/architecture.md`
Process layout + Live CLI event stream, `AGENTS.md` repo layout (no
`godot/` directory).
