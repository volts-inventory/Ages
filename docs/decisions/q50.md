## Q50 — Inter-civ knowledge transmission mechanism
**Status:** Decided (M4-minimal: linguistic-distance + age decay
+ flat tier factor; cultural-distance and per-relation tier
defer to v2 once Q24 + per-artifact persistence land)
**Context:** When a successor civ encounters knowledge left by a
collapsed predecessor, comprehension is gated by linguistic
distance, cultural distance, artifact age, and persistence tier.
The combining function and weights need definition. M3 has the
linguistic-distance primitive (NameGrammar atoms can be set-
compared); cultural-distance needs Q24 cosmology; per-artifact
tier needs an artifact infrastructure that doesn't exist yet.

**Decision:** Q50-minimal v1 ships a deterministic comprehension
score combining the available factors:

```
comprehension = (1 − linguistic_distance) × age_decay × tier_factor
              × (1 − cultural_distance)   // stub = 0 for now
```

- `linguistic_distance(a, b) = 1 − |atoms_a ∩ atoms_b| / |atoms_a ∪ atoms_b|`
  over the two civs' `NameGrammar.atoms` sets. Same strategy + same
  atoms → distance 0; different strategies share no atoms → 1.
- `age_decay = exp(−age_ticks / DECAY_CONSTANT_TICKS)` with
  `DECAY_CONSTANT_TICKS` placeholder under Q28. Year-meant: 1000
  years e-fold (Q67 scales the constant ×12 for the month-grained
  tick unit; the year-equivalent decay rate is preserved).
- `cultural_distance = 0` for v1 (Q24 cosmology not yet wired).
  Once Q24 lands, plug in cosmology-axis distance.
- `tier_factor = 0.7` flat — M4 v1 treats every relation as if
  inherited via T1 proto-symbolic artifacts. Per-artifact tier
  (T0 oral / T2 written / T3 mass-distributed) ships with the
  artifact-infrastructure follow-up.

A relation transmits when `comprehension > TRANSMIT_THRESHOLD =
0.15`. The threshold is tuned for the M4-minimum atom-pool sizes:
same-strategy civs reach `comprehension ≈ 0.2–0.4` typically;
cross-strategy civs stay at `0` because `linguistic_distance = 1`
zeros the term. Transmitted relations land on the successor's first-figure
hypothesizer's `confirmed` set with the predecessor's form +
params + residual; confidence is multiplied by `comprehension`
to reflect the partial recovery.

**Wiring.** After Q48 founds a successor, sim/core walks the
predecessor civ's figures' confirmed relations (dedupe by
`relation_id` taking the highest-confidence version), runs
the comprehension formula, and injects the survivors into the
new civ's first figure's hypothesizer. A
`KnowledgeTransmitted{tick, source_civ_id, dest_civ_id,
dest_figure_id, relation_id, source_form, comprehension_q32}`
event fires per transmitted relation.

**Why this satisfies M4 acceptance.** "A run that spans multiple
civs separated by a dark age, with the second civ partially
recovering knowledge from the first via inherited artifacts."
Q49 collapses, Q48 founds, Q50-minimal partially-transmits a
fraction of the predecessor's confirmed relations. The fraction
shrinks with linguistic drift (a successor with a different name
strategy understands less) and with age (a long dark age loses
more knowledge).

**Deferred to v2:**
- Cultural-distance term: Q24 cosmology axes.
- Per-artifact tier (Q30 persistence ladder per relation):
  artifact infrastructure not yet authored. The Civ would need
  per-relation `persistence_tier` and an "artifacts left in
  region" registry surviving across collapse.
- Multi-figure transmission targeting: v1 dumps everything into
  the first figure for simplicity. v2 distributes by figure
  interest / charisma / tier.

**Lives in:** `sim/civ/transmission.rs`, `sim/core::CivLifecycle`
post-founding, `protocol::KnowledgeTransmitted`.
