## Q20 — Milestone structure
**Status:** Decided (rewritten by Q39 + Q40, expanded by Q47)
**Context:** The split has been rewritten multiple times: original was
"M1 Earth-only, M2 variability"; later added event-log persistence to
M1; Q32 grew M1 with quantitative discovery; Q39 + Q40 dropped UI and
pivoted to a physics engine; Q47 elevated species to the persistent
unit and added civ lifecycle to M4.
**Decision:** Eight milestones (M0–M6 with M1 split into M1a + M1b
per Q41). Q51 dropped LLM from M6.
- **M0** — Scaffold + protocol JSON schemas + RNG + deterministic
  arithmetic strategy (Q36) wired in + CI determinism-test framework.
- **M1a** — Physics: mechanics + gravity + fluids + heat. Spatial
  grid; deterministic continuous simulation; snapshots + events from
  day one; live CLI stream.
- **M1b** — Physics: chemistry + simplified EM. Cross-couples with
  M1a substrate; emergent fire / lightning / phase boundaries.
- **M2** — Recognition layer + civ scaffold: pattern recognition turns
  physics state into recognized phenomena; cohort-based pop in default
  species; recognition-template library lands (~25 entries).
- **M3** — Discovery + species + named figures + first civ: species
  derivation, first civ founded inside the species, quantitative
  discovery pipeline (Q32) fitting against physics outputs, sensorium-
  extending tech, population dynamics, cultural hooks as no-ops.
- **M4** — Culture mechanics + civ lifecycle: axes, drift, holidays,
  language ladders, cultural-influence hooks fill in. Civ founding
  triggers, collapse triggers, artifact persistence across civ
  boundaries, inter-civ knowledge transmission with comprehension
  decay, successor-civ formation.
- **M5** — Concurrent multi-civ: multiple civs alive at once. Contact,
  knowledge diffusion, conflict.
- **M6** — Post-run report (no LLM per Q51): markdown report with
  templated prose where prose helps.
**Lives in:** `PLANNING.md` Milestones.
