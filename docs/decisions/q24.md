## Q24 — Cosmology drift dynamics
**Status:** Decided (M4: 5 named axes + per-event push-vectors
+ charisma-scaled magnitudes + Civ::dogmatism derivation)
**Context:** Culture has five orthogonal axes that drift with
events. Q23 ships the discovery-pipeline hooks (Q9 / Q25 / Q26)
as no-op stubs; this Q decides what the axes are, what events
push them, by how much, and how civ_dogmatism (referenced in
Q9 / Q26 formulas) gets derived.

**The five axes** (each per-civ scalar in `[-1, 1]`):

1. `Empirical` — `+1` trusts observation, `-1` trusts tradition.
2. `Communitarian` — `+1` collective decision-making, `-1`
   individualist.
3. `Reformist` — `+1` embraces refinement / new forms, `-1`
   conservative.
4. `Mystical` — `+1` favours mystical explanations, `-1`
   naturalist.
5. `Hierarchical` — `+1` stratified, `-1` egalitarian.

Civs found at `[0, 0, 0, 0, 0]` (unbiased) and drift over their
lifetime. Successor civs do *not* inherit cosmology in v1
(they're a fresh founding band; cosmology is civ-scoped).

**Per-event push-vectors** (scaled by `magnitude`):

| Event | Push (Empirical, Communitarian, Reformist, Mystical, Hierarchical) | Magnitude |
|---|---|---|
| `RelationConfirmed` | (+0.05, 0, +0.02, −0.04, 0) | × figure.charisma |
| `RefinementProposed` | (+0.02, 0, +0.04, 0, 0) | × figure.charisma |
| `RefinementConfirmed` | (+0.04, 0, +0.06, −0.02, 0) | × figure.charisma |
| `RefinementRejected` | (−0.02, 0, −0.04, +0.04, +0.02) | × figure.charisma |
| `CivCollapsed` | (0, +0.10, −0.10, +0.15, +0.05) | × 1.0 |

After each push, components are clamped to `[-1, 1]`. The
collapse vector is fixed-magnitude because the event affects
the entire civ irrespective of any one figure's charisma —
apocalyptic events drive entire populations toward mysticism +
hierarchy + tradition (a real-world pattern: collapses produce
mystery cults and authoritarianism).

**`Civ::dogmatism`** (read by Q9 / Q26):

```
dogmatism = clamp(|cosmology|_2 / sqrt(5), 0, 1)
```

L2-norm of the cosmology vector divided by `sqrt(5)` (the
maximum possible norm when every axis = ±1). Strong cosmology
in any direction → high dogmatism; an unbiased civ has 0
dogmatism. Captures "the more strongly held a cosmology, the
more it suppresses heretical hypotheses."

**`CosmologyShifted`** event emitted when L2 distance from the
last-emitted cosmology vector crosses `0.20` (don't spam tiny
deltas). Carries `(tick, civ_id, axes: [i64;5] in Q32.32 raw
bits, magnitude_q32: i64, dogmatism_q32: i64)`.

**Q9/Q25/Q26 wiring** (replacing M3 stubs):

- **Q9 `allow_observation(template_id, civ)`**: returns `false`
  with probability `dogmatism × cosmology_aligned_against(template)`,
  where the alignment term is a per-template tag (e.g. the fire
  template aligned-against high `Mystical`). M4-min stub: always
  true (the observation-rejection path needs Q57 settlement-pop
  to pick which cells get suppressed; deferred). Hook now returns
  the dogmatism scalar so callers can implement when ready.
- **Q25 `focus_weight(relation_id, civ)`**: returns weight in
  `[0, 1]`; high `Reformist` boosts novel-form relations, high
  `Empirical` boosts the most-observed candidate. Used by the
  hypothesizer's per-tick attempt-period scaling (M4-min: hook
  returns the cosmology-derived value but the hypothesizer
  doesn't yet vary cadence per relation; consumer lands when
  Q57/M4 v2 wires it).
- **Q26 `suppress_confidence(form, params, civ)`**: returns a
  multiplier in `[0, 1]` that's applied to fit confidence
  before the `≥ exp(-1)` confirmation check. Heretical forms
  (high cosmology distance to the civ's preferred form-tag set)
  in dogmatic civs accumulate confidence at a fraction of the
  base rate. **This one IS wired in M4-min** — fit pipeline
  multiplies confidence by the hook output before is_confirmed.
