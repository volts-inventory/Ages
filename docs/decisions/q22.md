## Q22 — Population scaling with resources and tech
**Status:** Decided (refined by Q47)
**Context:** Cohort-based pop dynamics initially had no coupling to
resources or tech. Q47 then made the population species-wide rather
than per-civ.
**Decision:** Population is owned by the species; each cohort carries
a `civ_membership` tag (a civ ID or `stateless`). Carrying capacity =
base_capacity(biome, regional substance inventory, equilibrium
biological-stock from physics) × tech_multiplier. The tech multiplier
applies to the civ that holds the region; stateless regions use the
hunter-gatherer baseline (multiplier = 1.0). Tech multipliers from
cultivation, domestication, irrigation, storage, sanitation, industrial
agriculture stack multiplicatively within a civ. Births / deaths /
migration tied to food security = 1 - max(0, pop / capacity - 1). No
hard ceiling; floor at zero (species extinction). Couples back to
discovery via larger pop → more named figures per active civ.
**Lives in:** `docs/civ.md` Population section.
